GitHub is a Git repository web-based hosting service which offers all of the functionality of Git as well as adding many of its own features. Unlike Git which is strictly a command-line tool, Github provides a web-based graphical interface and desktop as well as mobile integration. It also provides access control and several collaboration features such as wikis, task management, and bug tracking and feature requests for every project.

GitHub offers both paid plans for private repositories, and free accounts, which are usually used to host open source software projects. As of 2014, Github reports having over 3.4 million users, making it the largest code host in the world.

GitHub has become such a staple amongst the open-source development community that many developers have begun considering it a replacement for a conventional resume and some employers require applications to provide a link to and have an active contributing GitHub account in order to qualify for a job.

Dribbble is a community of designers answering that question each day. Web designers, graphic designers, illustrators, icon artists, typographers, logo designers, and other creative types share small screenshots (shots) that show their work, process, and current projects.

Dribbble is a place to show and tell, promote, discover, and explore design.

Dribbble began as a side project and is now a tiny, bootstrapped and profitable company helping the world's design talent share their creations and get hired. Dribbble has become a go-to resource for discovering and connecting with designers around the globe.

Twitter is an online social networking and microblogging service that enables users to send and read short 140-character text messages, called "tweets".

Registered users can read and post tweets, but unregistered users can only read them. Users access Twitter through the website interface, SMS, or mobile device app. Twitter Inc. is based in San Francisco and has offices in New York City, Boston, Austin and Detroit.



API
HTML
<div class="ui tab" style="min-height: 300px;"> <div class="nav"> <div class="item selected" data-index="github">Github</div> <div class="item selected" data-index="dribbble">Dribbble</div> <div class="item selected" data-index="twitter">Twitter</div> </div> <div class="content"> <section data-index="github"></section> <section data-index="dribbble"></section> <section data-index="twitter"></section> </div> </div>

Ajax页面的加载, 添加data-page="url"属性

Constructor

$( selector ).tab( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.tab" ).tab( { duration: 500 } );
Options

rule - "data-index"

页签和内容映射


class4loading - "loading"

Ajax加载内容时作用于页签


class4error - "error"

Ajax加载失败时作用于页签


ripple - true

页签点击大波浪


duration - 300

动画时间, 单位毫秒


onSelect - null

取值为function时, 页签选择时执行该方法, 取值为Hash对象时, 通过data-index执行对应回调

$( ".ui.tab" ).tab( { onSelect: { "github": function( tab, settings ) { console.log( "Github" ); }, "dribbble": function( tab, settings ) { console.log( "Dribbble" ); } } } );
Method

tab.add( items )

items - Object/Array

新增一个或多个tab

{ name - 新增页签显示, 取值不限, 可以为 HTML index - 页签索引值 render - 内容渲染, 返回内容 immediate - 是否立即加载内容, 默认 false }

tab.remove( index )

index - String

移除指定tab


tab.getTab( index )

index - String

获取指定tab的DOM


tab.active( index )

index - String

选中对应tab


tab.isActive( index )

index - String

判断对应tab是否被选中


tab.disabled( [index] )

index - String/Array

禁用tab, 参数空时禁用所有tab


tab.enabled( [index] )

index - String/Array

取消禁用, 参数空时取消所有已禁用tab
