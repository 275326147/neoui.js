API
HTML
<div class="ui slide open" style="width: 800px;"> <header class="title">Header</header> <div class="content"> <h1>#CONTENT#</h1> </div> </div>


Constructor

$( selector ).slide( [options] )

selector - 选择器或 DOM 元素

options - 参数设置

        
        $( ".ui.slide" ).slide();


Options

init - null

初始化时执行


class4open - "open"

展开时样式


selector4trigger - ".title:first"

匹配触发器


selector4content - ".content:first"

定位内容区域


onOpen - null

内容区域展开时执行


onClose - null

内容区域收起时执行




Method

slide.open()

展开内容区域


slide.close()

关闭内容区域


slide.open()

展开或收起内容区域
