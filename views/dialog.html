Example
NOTE: 服务随便找的, 可模模糊查询

API
HTML
<div class="ui dialog"> <input class="ui text front" type="text" placeholder="Type to search" /> <input class="ui text hint" type="text" tabindex="-1" /> <i class="icon search"></i> </div>
Constructor

$( selector ).dialog( options )

selector - 选择器或 DOM 元素

options - 参数设置

$( ".ui.dialog[name=mrQuery]" ).dialog( { multiple: true, ajaxOptions: { serviceName: "queryPorject", moduleName: "gridElement_publicfind", params: { "functionId": "12115", "popUpProgramName": "mrQueryMrNo" }, title: "Mr No.", onOk: function( items ) { var text = []; for ( var i = (items || []).length; --i >= 0; text.unshift( items[ i ][ "mrNo" ] ) ); this.val( text.join( "" ) ); }, columns: [ { id: "mrno", field: "mrno", name: "MR NO.", filter: true, sortable: true, width: 600 } ] } } );
Options

css - { width: 800 }

弹出框样式设置


closeByDocument - true

点击弹出框外面, 关闭窗口


forceFitColumns - true

强制对齐所有列, 消除横向滚动条


key - find

搜索框name值, 搜索时会将该值传入后台


uniqueKey - rr

服务所返回数据的索引键, 比如rownum


pagingInfo - { pageSize: 100, pageNum: 0 }

分页参数, 详见 SlickGrid+ Pager


autoSearch - true

打开弹出框后默认查询, 设置为false后, 默认不查询


increment - true

为true时为增量分页


fastMode - false

设置为true时, 排序, 过滤操作交由浏览器进行本地排序或过滤, 为false时, 由服务器端处理


switcher - true

是否显示切换按钮


multiple - false

是否多选


gridOptions - null

表格参数参数, 见 Data Grid


selector4input - "input:first"

匹配对应输入框, 用于高级定制


selector4trigger - ".icon:first"

触发器, 取值选择器或 DOM 元素


autoComplete - false

接收 AutoComplete 控件参数, 详见 AutoCompelte


onOk - null

点击 Ok 按钮时调用


onClose - null

点击 Cancel 按钮时调用

Method

dialog.val( [value] )

value为可选, 不为空时, 设置对应值到输入框, 为空时返回对应输入框文本值, 对于特殊值, 请添加自定义属性或对 DOM 进行数据绑定


dialog.disabled()

禁用控件, 同 DOM 节点添加disabled="disabled"


dialog.enabled()

启用控件
