
# Neoui.js
>模块化，高扩展性，组件齐全，接口一致，自主开发，适合多种应用场景

### Install
```
npm install
```

### Build
```
grunt
```

### Have a try
```
$ clone https://github.com/trazyn/neoui.js.git
$ cd neoui.js/
$ python -m SimpleHTTPServer

http://127.0.0.1:7000
```

>部分数据无法直接访问, 需 nginx 反向代理, 请使用 Chrome 浏览器, 暂时未做其他浏览器兼容
